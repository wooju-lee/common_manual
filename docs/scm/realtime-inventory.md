# 📍 실시간 재고 조회

> 매장·창고별 현재 재고 현황을 실시간으로 조회하는 기능입니다.
>
> 경로 : `Inventory > Realtime Inventory`

## AS-IS → TO-BE

| 구분 | AS-IS (Netsuite ERP) | TO-BE (IIC BO) |
|:---:|---|---|
| **재고 조회** | Netsuite에서 Location별 개별 조회 | IIC BO에서 매장·창고별 통합 조회 |
| **가용 재고** | 보유 수량만 표시 | 보유·예약·조정대기·가용 재고를 구분하여 표시 |
| **검색** | 제한적 검색 | 브랜드/법인/매장/카테고리/상품코드/바코드 통합 검색 |

:::tip 핵심 변경점
- **가용 재고 구분** — 단순 보유 수량이 아닌, 실제로 사용 가능한 가용 재고를 별도 표시
- **통합 검색** — 하나의 화면에서 브랜드, 법인, 매장, 카테고리, 상품명, 바코드 등 다양한 조건으로 검색
:::

---

## 재고 수량 항목

| 항목 | 설명 | 비고 |
|:---:|---|---|
| **보유 수량** | 해당 창고에 실제 보유한 총 수량 | |
| **예약 수량** | 출고 요청 승인 대기 중인 수량 | 출고 요청 시 자동 증가 |
| **조정 대기** | 재고 조정 승인 대기 중인 수량 | 조정 등록 시 자동 증가 |
| **가용 재고** | 실제 사용 가능한 수량 | = 보유 - 예약 - 조정대기 |

:::info 가용 재고 계산
**가용 재고 = 보유 수량 − 예약 수량 − 조정 대기 수량**

예) 보유 100개, 출고 요청 20개, 조정 대기 5개 → 가용 재고 = **75개**
:::

## 조회 방법

| Step | 처리 내용 |
|:---:|---|
| **1** | `Inventory > Realtime Inventory` 메뉴 진입 |
| **2** | 검색 조건 설정 |
| **3** | 조회 실행 |
| **4** | 필요 시 엑셀 다운로드 |

### 검색 조건

| 조건 | 설명 | 필수 여부 |
|---|---|:---:|
| 브랜드 | 브랜드 선택 | 선택 |
| 법인 | 법인 코드 선택 | 선택 |
| 매장 | 매장 코드 선택 (복수 가능) | 선택 |
| 창고 | 창고(Location) 선택 (복수 가능) | 선택 |
| 카테고리 | 1차/2차 카테고리 선택 | 선택 |
| 검색어 | 법인명, 매장명, 상품코드, 상품명, 바코드 통합 검색 | 선택 |

### 상세 조회 (매장·창고 단위)

특정 매장·창고를 클릭하면 해당 위치의 상품별 재고를 상세 조회할 수 있습니다.

| 옵션 | 설명 |
|---|---|
| **전체 상품 포함 OFF** (기본) | 재고가 있는 상품만 표시 |
| **전체 상품 포함 ON** | 해당 브랜드의 모든 상품 표시 (재고 0인 상품은 0으로 표시) |

## 정렬 가능 항목

보유 수량, 예약 수량, 조정 대기 수량, 가용 재고 각 항목으로 오름차순/내림차순 정렬이 가능합니다.

---

## FAQ

### Q. 다른 매장의 재고도 볼 수 있나요?

> 본인에게 권한이 부여된 매장(브랜드/법인/매장 범위) 내에서만 조회 가능합니다.
> 권한 범위 밖의 매장 재고는 표시되지 않습니다.

---

### Q. 가용 재고가 마이너스가 될 수 있나요?

> 정상적인 상황에서는 마이너스가 되지 않지만, 시스템 간 데이터 동기화 지연 등으로 일시적으로 발생할 수 있습니다.
> 이 경우 재고 조정을 통해 정정해 주세요.

---

### Q. 재고 조회 시점의 데이터인가요?

> 네. 조회 시점의 실시간 재고 데이터입니다.
> 다른 사용자가 동시에 출고/입고/이동을 처리하면 수량이 달라질 수 있으므로, 중요한 작업 전에는 새로고침 후 확인해 주세요.
