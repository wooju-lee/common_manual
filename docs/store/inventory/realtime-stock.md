# 📍 실시간 재고 조회

> 현재 시점의 전산 재고를 스토어, 로케이션, 아이템별로 실시간 확인할 수 있는 메뉴입니다.
>
> 경로 : <span className="path-badge">IIC BO > Inventory > 재고 리스트</span>

---

## 👉 언제 사용하나요?

- 고객 응대 시 **특정 제품의 재고 보유 여부**를 빠르게 확인하고 싶을 때
- 매장 내 **로케이션별 재고 현황**을 파악하고 싶을 때
- 출고/조정 진행 중인 제품의 **실제 운용 가능 수량**을 확인하고 싶을 때

---

## 1. 재고 조회 방법

조건 선택 또는 텍스트 직접 입력을 통해 재고를 조회할 수 있습니다.

| 조건 | 설명 |
|---|---|
| **브랜드** | 담당 브랜드 기준 기본값 설정 |
| **법인** | 법인 선택 시 해당 스토어 목록 조회 가능 |
| **스토어** | 담당 스토어가 설정된 경우 해당 스토어만 조회. 멀티 선택 가능 |
| **로케이션** | 스토어 내 로케이션별 조회. 멀티 선택 가능 |
| **제품 카테고리** | 카테고리 1 선택 시 카테고리 2 활성화 |
| **텍스트 검색** | 법인 코드, 법인명, 스토어 코드, 스토어명, 제품 코드, 제품명, 바코드 직접 입력 |

## 2. 실시간 재고 수량 확인 방법

조회된 리스트에서 아래 항목을 확인할 수 있습니다.

| 항목 | 의미 | 참고 |
|---|---|---|
| **총 재고 수량** | 해당 로케이션 내 제품의 전체 수량 | — |
| **가용 재고** | 실제 판매·이동에 활용할 수 있는 수량 | 총 재고 - 출고 대기 - 조정 대기 |
| **출고 대기** | 출고 등록은 되었으나 아직 완료되지 않은 수량 | 입고 완료 시 제외 |
| **조정 대기** | 승인 대기 중인 재고 조정 수량 | 승인 완료 시 반영 |

- <mark>고객에게 재고 안내 시에는 **가용 재고** 기준으로 확인해 주세요. 출고 대기/조정 대기 수량은 실제 운용이 불가한 수량입니다.</mark>
- 조회된 데이터는 **엑셀 다운로드**하여 활용할 수도 있습니다.

## ❓ FAQ

> **Q. 총 재고 수량과 가용 재고가 다른 이유가 뭔가요?**
>
> A. 출고 등록 또는 재고 조정이 진행 중인 수량은 <u>가용 재고</u>에서 제외됩니다.
>
> 실제 운용 가능한 수량은 가용 재고를 기준으로 확인해 주세요.

> **Q. 특정 제품의 재고를 빠르게 찾고 싶어요.**
>
> A. 텍스트 검색 필드에 <u>제품명, 제품 코드 또는 바코드</u>를 직접 입력하면 빠르게 조회할 수 있습니다.

> **Q. 엑셀 다운로드는 전체 데이터인가요?**
>
> A. 현재 검색 조건에 맞는 <u>조회된 데이터</u> 기준으로 추출됩니다.

</div>
